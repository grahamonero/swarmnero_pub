Run with Pear Runtime Recommended

Swarmnero is built on Pear Runtime — the peer-to-peer app platform from Holepunch. Install Pear once, then launch Swarmnero with a single command. Updates arrive automatically over the P2P network.

  1. Install Pear Runtime (one-time, ~30 seconds): 
    npm i -g pear
    Don't have Node.js? See the full Pear install guide.
  2. Launch Swarmnero: 
    pear run pear://<key-coming-soon>

Why Pear? No installer warnings. Auto-updates. One runtime powers many P2P apps. Read the full guide →

Standalone download

Don't want to install Pear? Grab a native installer for macOS, Windows, or Linux. These are Electron wrappers that bundle Pear and launch Swarmnero on first run.

Unsigned builds. These installers are not code-signed (no Apple Developer ID, no Microsoft Authenticode cert). Your OS will show a security warning. This is expected, not a problem with the build:
  • macOS: After opening the .dmg, right-click the Swarmnero app and choose Open (don't double-click). Confirm the dialog once; future launches work normally. On newer macOS, you may also need to run xattr -d com.apple.quarantine /Applications/Swarmnero.app in Terminal.
  • Windows: SmartScreen will say "Windows protected your PC." Click More info → Run anyway.
  • Linux: Mark the AppImage executable (chmod +x) or install the .deb with sudo dpkg -i.
If you'd rather skip the warnings entirely, use Pear Runtime.

What makes it different

Fully peer-to-peer

No central servers. Your posts replicate directly between devices over the Hyperswarm DHT.

Monero tipping

Built-in XMR wallet. Tip posts and replies. No custodians, no middlemen.

Paywalled posts

Encrypt premium content, gate it behind a Monero payment. Fully automated unlock — no manual approvals.

Private DMs

End-to-end encrypted direct messages between mutual follows. X25519 + crypto_secretbox.

No algorithms

Chronological timelines. Follow who you want. Discover via friends-of-friends and hashtag search.

Your keys, your identity

Ed25519 identity stored locally. Optional password encryption. Export and import at any time.
