Install

Swarmnero runs on Pears — the peer-to-peer app platform from Holepunch. Install Pears once, then launch Swarmnero with a single command. Updates arrive automatically over the P2P network.

  1. Install Pear Runtime (one-time, ~30 seconds): 
    npm i -g pear
    Don't have Node.js? See the full install guide.
  2. Launch Swarmnero: 
    pear run pear://y3m1mm1xxrqj9jz5jpicc57rt4dem5t5g9s78j9riqft94u4j6jy

What makes it different

Fully peer-to-peer

No central servers. Your posts replicate directly between devices over the Hyperswarm DHT.

Monero tipping

Built-in XMR wallet. Tip posts and replies. No custodians, no middlemen.

Paywalled posts

Encrypt premium content, gate it behind a Monero payment. Fully automated unlock — no manual approvals.

Private DMs

End-to-end encrypted direct messages between mutual follows. X25519 + crypto_secretbox.

No algorithms

Chronological timelines. Follow who you want. Discover via friends-of-friends and hashtag search.

Your keys, your identity

Ed25519 identity stored locally. Optional password encryption. Export and import at any time.
