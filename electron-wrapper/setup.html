Welcome to Swarmnero

Swarmnero runs on Pear Runtime — a peer-to-peer app platform from Holepunch. Pear is not installed on this computer yet.

1

Install Node.js (if you don't already have it)

Download the LTS version from nodejs.org.

2

Install Pear Runtime

Open Terminal (or Command Prompt) and run:

npm i -g pear
3

Relaunch Swarmnero

Once Pear is installed, click Recheck below — or quit and reopen Swarmnero.
